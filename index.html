Powerful Features

Live Presence

Instantly detects when a user goes Online, Offline, enters Studio, or starts playing a game.

Deep Join Links

Generates direct "Join Game" links that bypass the server list and launch you straight into their server.

Server Stats

Automatically fetches public server Ping, FPS, and Player Count when available.

Hybrid Commands

Full support for both Slash Commands (/) and customizable Prefix Commands (e.g. !).

Private Detection

Intelligently detects when a user has joins disabled or is in a private server.

Metadata Logs

Tracks friend list changes (adds/removes) and avatar updates in real-time.

Bot Commands

Tracking

/list add Add users to the tracking list. Supports comma-separated lists.
/list remove Remove a user from the tracking list.
/list show View all currently tracked users.

Configuration

/setprefix Change the bot's command prefix (e.g. !, ?, .).
/setchannel Direct alerts to specific 'events' or 'logs' channels.
/sync Fix missing slash commands. Use global_clear:True to fix duplicates.

System

/health Check system uptime and API latency.
/showlogs View or download the internal log file.
/mancheck Manually force a presence check on a specific user.

How to Setup

RBXStalker V2 is open-source and self-hosted. You control the data.

# 1. Clone the repository

git clone https://github.com/StalkRoblox/RBXStalker-V2.git


# 2. Install requirements

pip install -r requirements.txt


# 3. Configure .env file with your Discord Token

DISCORD_TOKEN=your_token_here


# 4. Run the bot

python main.py
